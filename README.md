## Overview

Manage a Starlink satellite internet connection with optional power control.

<br/>

## Configuration

| Setting | Description | Default |
|---------|-------------|---------|
| **Starlink IP Address** | IP address of the dish | `192.168.1.1` |
| **Power Pin** | Digital output for power control | `None` |

<br/>
## Integrations

This is a standalone app with no dependencies on other Doover apps.
